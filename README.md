# ASML-TSMC-Valuation-Analysis

This notebook contains the valuation analysis and data visualizations for the paper "Beyond the Picks and Shovels: Why ASML and TSMC Represent More Durable AI Infrastructure Investments Than Fabless Designers" (Kadam, 2026). It pulls live P/E and market cap data via yfinance and generates comparative stock price trajectory charts for Cisco (1995-2002) and NVIDIA (2019-2026).
